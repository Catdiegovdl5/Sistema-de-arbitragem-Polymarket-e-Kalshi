# Bot de Arbitragem Kalshi & Polymarket

Bem-vindo ao seu **Sistema Avançado de Arbitragem Cross-Platform**. Este sistema utiliza inteligência artificial (Processamento de Linguagem Natural) para cruzar milhares de mercados ao vivo entre a Kalshi e Polymarket em busca de oportunidades seguras.

A interface gráfica de alta performance permite que você monitore tudo do conforto do seu navegador, com atualizações em tempo real!

## Como Iniciar

Siga estes 2 passos simples para ver a mágica acontecer na sua tela:

### 1. Instalar Dependências (Apenas na Primeira Vez)
Dê um duplo clique no arquivo **`Instalar_Dependencias.bat`**. 
Ele vai abrir uma tela preta instalando tudo o que o sistema precisa automaticamente. Quando ele disser "INSTALACAO CONCLUIDA COM SUCESSO!", você pode fechar a tela.

### 2. Inicie o Sistema (Motor + Painel Visual)
Dê um duplo clique no arquivo chamado **`start_all.py`**. 
Isso abrirá uma janela preta (terminal) carregando todos os 5.000 mercados nas corretoras e conectando o cérebro do robô.
*(Deixe essa tela preta aberta! Ela é o motor do seu carro).*

### 3. Abra o Painel de Controle
Abra o seu navegador de internet preferido (Google Chrome, Edge, Safari) e acesse o endereço abaixo:

**[http://localhost:5000](http://localhost:5000)**

Pronto! Você agora está conectado à matriz.
A interface visual vai carregar e o status mudará para **ONLINE** em verde assim que a conexão com a corretora for firmada. As tabelas se auto-atualizarão a cada 1 segundo.

---

*Nota Técnica: O sistema é composto por duas camadas profissionais separadas (uma API de alta velocidade em FastAPI na porta 8000 e um painel Flask/BlackDashboard na porta 5000). Tudo já está configurado para se comunicar perfeitamente.*
